# Skill: Git Commit en Español

Cuando escribas un mensaje de commit, SIEMPRE usá español argentino.

Seguí el formato Conventional Commits pero con la descripción en español:
- feat: descripción en español
- fix: descripción en español
- chore: descripción en español
- refactor: descripción en español

## Ejemplos correctos
- feat: agregar validación de stock en el formulario de venta
- fix: corregir cálculo de precio con descuento
- chore: actualizar dependencias de React

## Ejemplos INCORRECTOS (nunca hagas esto)
- feat: add stock validation
- fix: fix price calculation bug
